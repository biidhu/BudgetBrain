# Project Title

BudgetBrain - Smart Finance Manager

---

# Problem Statement

Many individuals and small groups struggle to track their personal finances effectively. Without a simple, accessible tool, people lose track of income, expenses, and budgets across different categories. Existing financial tools are often too complex or require expensive subscriptions.

Our project solves this by providing a free, intuitive personal finance management web application that allows users to track transactions, categorize expenses, and set budgets - all in one place.

---

# Target Users

- University students managing budget
- Small business owners tracking expenses
- Individuals wanting to control personal spending
- Groups sharing finances

---

# Project Goal

The system will allow users to:
- Register and login securely
- Add income and expense transactions with categories
- View dashboard with balance, total income, and total expenses
- Set and manage budgets by category
- Visualize spending patterns through charts

---

# Core Features (First Version)

- User registration and authentication (JWT)
- Transaction management (add income/expenses)
- Dashboard with financial summary
- Category-based budget tracking
- Visual charts for spending analysis

---

# Demo Scenario

Step 1: User opens the BudgetBrain website

Step 2: User registers or logs in

Step 3: User adds income transaction (e.g., salary)

Step 4: User adds expense transaction (e.g., food)

Step 5: User views updated dashboard with balance and charts

Step 6: User sets a budget for a category

---

# Minimum Viable Product (MVP)

The MVP will allow a user to:
1. Register and login
2. Add income and expense transactions
3. View a dashboard showing total balance, income, and expenses
4. Set category budgets and see budget progress
